# IT Support Ticketing System

## Overview

This project demonstrates hands-on experience using a ticketing system (osTicket) to manage and resolve common IT support issues.

A local support environment was configured to simulate real-world workflows, including ticket intake, troubleshooting, communication with users, and SLA-based response handling.

---

## What I Implemented

- Configured osTicket in a local environment using XAMPP
- Created and categorized support tickets using custom help topics
- Responded to user issues with clear, step-by-step troubleshooting
- Documented internal notes to track investigation and resolution steps
- Applied SLA policies to simulate response time expectations
- Managed ticket workflow from intake to resolution
